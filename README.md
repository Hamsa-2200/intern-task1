# Internship Task 1 – Bootstrap 5 Website

## Project Overview
This project was developed as part of the internship task. The objective was to build a responsive multi-page website using Bootstrap 5 components without copying complete templates.

## Pages Created
- Home Page (index.html)
- About Page (about.html)
- Contact Page (contact.html)

## Bootstrap Components Used
- Navbar
- Container
- Grid System
- Cards
- Buttons
- Forms
- Footer

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5 (via CDN)
- GitHub for hosting

## How I Built This Project
1. Created project folder and HTML files.
2. Connected Bootstrap using CDN.
3. Designed the Home page with navbar, hero section, cards, and footer.
4. Created About page with project details.
5. Built Contact page with Bootstrap form.
6. Tested responsiveness.
7. Uploaded project to GitHub and deployed using GitHub Pages.

## Challenges Faced
- Understanding Bootstrap grid system initially.
- Linking multiple pages correctly.
- Making sure navigation works properly.

## Learning Outcome
This project helped me understand how Bootstrap components work, how responsive layouts are structured, and how to deploy a static website using GitHub Pages.
